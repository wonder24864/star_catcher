# 管理员模块

用户管理、系统配置。Sprint 3（精简版）。

---

## US-027: 管理员用户管理

**As a** 管理员
**I want to** 管理系统中的用户
**So that** 我可以维护用户数据

**验收标准：**
- [ ] 查看所有用户列表（支持搜索、分页）
- [ ] 查看用户详情（角色、家庭组、错题统计）
- [ ] 禁用/启用用户账号
- [ ] 重置用户密码
- [ ] 不能删除用户（只能禁用）

---

## US-028: 管理员系统配置

**As a** 管理员
**I want to** 配置系统基础参数
**So that** 系统行为可以调整

**验收标准：**
- [ ] 配置AI识别参数（模型选择、温度等）
- [ ] 配置文件上传限制（大小、格式）
- [ ] 配置默认的求助Level策略
- [ ] 查看系统运行统计（用户数、错题数、AI调用次数）
