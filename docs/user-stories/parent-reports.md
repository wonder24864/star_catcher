# Parent Reports 用户故事

## US-041: 家长周报/月报

**As a** 家长
**I want to** 查看孩子在知识点维度的学习进步趋势和薄弱分析报告
**So that** 了解孩子的阶段性学习成效，识别需要重点关注的知识点

**验收标准：**
- [ ] 家长端新增"学习报告"页面，支持 7 天（周报）/ 30 天（月报）切换
- [ ] 报告概览卡片：新掌握 KP 数量、退步 KP 数量、新增错题 KP 数量、复习完成率
- [ ] 掌握趋势图：Recharts LineChart，按日统计 MASTERED 状态 KP 总数变化
- [ ] 薄弱知识点排行：Top 5 薄弱 KP（status = NEW_ERROR/CORRECTED/REGRESSED），显示学科 + 名称 + 尝试次数 + 最后一次错误时间
- [ ] 复习完成率卡片：时间窗口内已完成复习 / 应复习任务数
- [ ] 单个 KP 进展详情：点击薄弱 KP 可查看干预历史时间线（InterventionHistory）
- [ ] 数据范围由日期计算，不依赖 cron 预生成
- [ ] 家长只能查看自己家庭成员的学生报告（RBAC）
- [ ] 学生本人也可查看自己的报告

**边界条件：**
- 学生无任何 MasteryState 记录时：显示空状态"暂无学习数据"
- 时间窗口内无状态变更时：概览卡片全部为 0，趋势图为平线
- 复习完成率分母为 0（无调度）时：显示"暂无复习安排"
- 学生选择器：复用现有 useStudentStore

**性能要求：**
- weeklyReport / monthlyReport 查询 < 2s
- 页面首屏加载 < 3s（含图表渲染）
