# PWA 与国际化模块

PWA 安装与离线支持、中英双语切换。Sprint 3。

---

## US-029: PWA安装

**As a** 用户
**I want to** 把网站安装到手机主屏
**So that** 像使用App一样方便

**验收标准：**
- [ ] 浏览器显示"添加到主屏幕"提示
- [ ] 安装后以独立窗口（standalone）打开，无浏览器地址栏
- [ ] 应用图标和启动画面使用Star Catcher品牌设计
- [ ] 离线模式支持范围：错题列表（只读）、错题详情（只读）
- [ ] 离线时写操作（添加备注、求助等）显示提示："请连接网络后操作"
- [ ] 恢复连接后自动同步
- [ ] Service Worker 缓存策略：
  - 静态资源：Cache First
  - API请求：Network First，离线时返回缓存
  - 图片：Cache First，后台更新

## US-030: 中英文切换

**As a** 用户
**I want to** 切换界面语言为中文或英文
**So that** 我可以用熟悉的语言使用系统

**验收标准：**
- [ ] 设置页面可切换语言
- [ ] 所有静态文本支持中/英
- [ ] AI生成的内容根据当前语言设置生成对应语言
- [ ] 英语学科题目保持英文原文
- [ ] 数学公式使用KaTeX渲染（语言无关）
- [ ] URL带locale前缀（`/zh/...`, `/en/...`）
- [ ] 用户语言偏好保存到账号设置
